
# AI for Everyone: 
# Build Your Brand, Website, and Content Without Coding



## 
**Setup**
1. Create account here https://lovable.dev/  (Create free account, you can select 'Continue with Google')
2. Login to your account https://lovable.dev/

## 
**Ideation / Product / Brand**

I want to build a website for my wife that will showcase her professional and personal brand. 

##
**Use AI to create website with AI**

1. Visit linkedin
2. Copy name, headline, about, experience, education, licenses, volunteering, and recommendations
3. Paste it in a text editor (Sublime or Notepad)
4. Now create a prompt for chatgpt
5. I want to create a website with following details, help me create a prompt for loveable so it can create my website.  Note: this is personal website and I am female professional in Tech and Edtech space, so select colors and theme accordingly.  Here are my details "<paste the linkedin information>"
6. ... chatGPT will produce a prompt, copy the prompt

##
**Let's create website using AI**

1. Login to loveable and in chat section under 'What should we build, AI?  paste it.  and select 'Plan' in bottom corner, then hit 'up arrow'
2. Observe the plan (for the purpose of our session, we will accept what it says)


## 
**Let's publish to the test area (Staging)**

1. Select the blue button top-right (up arrow, or publish)
2. Change the 'Your website URL' to something you like
3. Continue x 4 time and then hit 'Publish'
4. ... it will show spinning symbol, once finished, you will see 'View app ->'
5. Note the domain name above it will be <something>.lovable.app  (copy and paste it in your notepad)

##
**Let's change email and linkedin**

1. On the left side you can say <what you want to change>, in my case (Change email from hello@sadiaasif.com to sadia@asif.ai)
2. Select Plan, Hit up arrow
3. It will say few things (just ignore) then it will show plan, for our purpose we will 'Approve'
4. It will do few things and then it will be done
5. Publish again, by selecting 'Publish' (top right corner) and then 'Update'
6. It will spin for few seconds and once it's stops, check the website

##
**Voilà**   _("there it is" or "here you go")_  Your site is ready!
